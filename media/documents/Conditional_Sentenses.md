A conditional sentence usually has two parts:

> **If + condition, result**

Example:

> If it rains, I will stay home.
>
> যদি বৃষ্টি হয়, আমি বাড়িতে থাকব।

---

# The 5 Main Types of Conditionals

## Zero Conditional (General Truth)

### Structure

**If + Present Simple, Present Simple**

### English

If you heat water to 100°C, it boils.

### Bangla

যদি তুমি পানি ১০০° সেলসিয়াস পর্যন্ত গরম করো, তাহলে তা ফুটে।

### Why?

This is always true. It is a scientific fact or general truth.

More examples:

* If you don't eat, you get hungry.

  * যদি না খাও, ক্ষুধা লাগে।

* If it rains, the ground gets wet.

  * বৃষ্টি হলে মাটি ভিজে যায়।

Think:

**Whenever X happens, Y happens.**

---

# First Conditional (Real Future Possibility)

### Structure

**If + Present Simple, will + Verb**

### English

If I study hard, I will pass the exam.

### Bangla

যদি আমি মনোযোগ দিয়ে পড়ি, আমি পরীক্ষায় পাস করব।

### Why?

The condition is possible and realistic.

More examples:

* If it rains tomorrow, I will stay home.

  * যদি কাল বৃষ্টি হয়, আমি বাসায় থাকব।

* If you call me, I will help you.

  * তুমি আমাকে ফোন করলে আমি সাহায্য করব।

Think:

**Possible future situation.**

---

# Second Conditional (Unreal Present/Future)

### Structure

**If + Past Simple, would + Verb**

### English

If I were rich, I would buy a Ferrari.

### Bangla

যদি আমি ধনী হতাম, আমি একটি ফেরারি কিনতাম।

### Why?

You are not rich now.

The situation is imaginary.

More examples:

* If I knew French, I would work in Paris.

  * যদি আমি ফরাসি জানতাম, আমি প্যারিসে কাজ করতাম।

* If I had wings, I would fly.

  * যদি আমার ডানা থাকত, আমি উড়তাম।

Think:

**Dreams, imagination, unlikely situations.**

---

### Special Note

We usually say:

✅ If I were

Not:

❌ If I was

Example:

* If I were you, I would study more.

বাংলা:

* আমি যদি তোমার জায়গায় হতাম, আরও পড়াশোনা করতাম।

This is a very common expression.

---

# Third Conditional (Unreal Past)

### Structure

**If + Past Perfect, would have + Verb₃**

### English

If I had studied harder, I would have passed.

### Bangla

যদি আমি আরও ভালো করে পড়তাম, আমি পাস করতাম।

### Why?

The event is already finished.

You cannot change the past.

More examples:

* If I had left earlier, I would have caught the bus.

  * যদি আমি আগে বের হতাম, বাস ধরতে পারতাম।

* If she had called me, I would have helped her.

  * যদি সে আমাকে ফোন করত, আমি তাকে সাহায্য করতাম।

Think:

**Regret about the past.**

---

# Mixed Conditional

Sometimes we mix past and present.

### Structure

**If + Past Perfect, would + Verb**

### English

If I had studied medicine, I would be a doctor now.

### Bangla

যদি আমি মেডিসিন পড়তাম, তাহলে এখন ডাক্তার হতাম।

### Why?

Past cause → Present result

Past:

* I didn't study medicine.

Present:

* I am not a doctor.

Another example:

* If I had learned English earlier, I would speak fluently now.

বাংলা:

* যদি আমি আগে ইংরেজি শিখতাম, এখন সাবলীলভাবে বলতে পারতাম।

---

# Quick Comparison

### First Conditional

Possible future

> If I study, I will pass.

আমি পড়লে পাস করব।

---

### Second Conditional

Imaginary present/future

> If I studied, I would pass.

আমি যদি পড়তাম, পাস করতাম।

(But maybe I won't study.)

---

### Third Conditional

Imaginary past

> If I had studied, I would have passed.

আমি যদি পড়তাম, পাস করতাম।

(But the exam is already over.)

# The Three Most Important Sentences

Memorize these:

### First Conditional

If I study hard, I will pass.

যদি আমি মনোযোগ দিয়ে পড়ি, আমি পাস করব।

---

### Second Conditional

If I were rich, I would travel the world.

যদি আমি ধনী হতাম, আমি পৃথিবী ভ্রমণ করতাম।

---

### Third Conditional

If I had studied harder, I would have passed.

যদি আমি আরও ভালো করে পড়তাম, আমি পাস করতাম।

# Memory Trick

| Type   | Formula                       | Meaning                 |
| ------ | ----------------------------- | ----------------------- |
| Zero   | If + Present, Present         | Fact                    |
| First  | If + Present, Will            | Real future possibility |
| Second | If + Past, Would              | Unreal present/future   |
| Third  | If + Past Perfect, Would have | Unreal past/regret      |

A simple shortcut:

* **Zero = Fact**
* **First = Maybe**
* **Second = Imagine**
* **Third = Regret**

If you master these four, you'll understand about 95% of conditional sentences used in speaking, writing, movies, books, and exams.
